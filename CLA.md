## How to sign

By submitting a pull request to the Project, You indicate agreement to this Agreement for that and
all subsequent Contributions.

For an explicit record, sign off each commit with a real name and email:

```sh
git commit -s -m "your message"
```

which appends:

```
Signed-off-by: Your Name <you@example.com>
```
